Sextortion
  1. Stop responding. Do not pay.
  2. Preserve evidence (screenshots, links, timestamps).
  3. Secure accounts: change passwords, enable 2FA.
  4. Report to platform/cyber cell and save reference IDs.
Cyberstalking
  1. Log incidents with date/time and handles.
  2. Block & tighten privacy settings.
  3. Disable location sharing; audit app permissions.
  4. Escalate to police/cyber cell if harassment continues.
